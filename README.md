# Python Study Sprint

Pythonのプログラミング言語能力を学ぶための、シンプルなStreamlit製学習アプリです。

## 機能
- 基本文法を短く学べるレッスンカード
- すぐに理解度を確認できる3問クイズ
- 実践課題と学習メモ
- 4週間の学習ロードマップ

## セットアップ
```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
streamlit run app.py
```

## 想定ユーザー
- Pythonをこれから学び始める人
- 文法の基礎を短時間で反復したい人
- 小さな学習アプリから学習習慣を作りたい人
